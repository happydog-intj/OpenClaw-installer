## 🚀 下一步：发布到 GitHub

### 步骤 1: 推送到 GitHub

```bash
cd ~/Documents/Github/OpenClaw-installer

# 如果还没有创建 GitHub 仓库，先创建
# 访问 https://github.com/new
# 仓库名: OpenClaw-installer
# 描述: OpenClaw 一键安装工具 - 图形化界面，零技术门槛

# 添加远程仓库（替换为你的用户名）
git remote add origin https://github.com/你的用户名/OpenClaw-installer.git

# 推送代码
git push -u origin master
```

### 步骤 2: 创建 GitHub Release

1. 访问你的 GitHub 仓库页面
2. 点击右侧 "Releases" → "Create a new release"
3. 填写信息：
   - **Tag**: `v0.1.0`
   - **Title**: `OpenClaw Installer v0.1.0 - 首个公开版本`
   - **Description**: 复制 `RELEASE_GUIDE.md` 中的 Release notes

4. 上传安装包：
   - 点击 "Attach binaries by dropping them here"
   - 将以下文件拖入：
     ```
     ~/Documents/Github/OpenClaw-installer/src-tauri/target/universal-apple-darwin/release/bundle/dmg/OpenClaw Installer_0.1.0_universal.dmg
     ```

5. 点击 "Publish release"

### 步骤 3: 更新 README 下载链接

Release 发布后，GitHub 会自动生成下载链接，格式如下：

```
https://github.com/你的用户名/OpenClaw-installer/releases/download/v0.1.0/OpenClaw%20Installer_0.1.0_universal.dmg
```

或使用 "latest" 链接（自动指向最新版本）：

```
https://github.com/你的用户名/OpenClaw-installer/releases/latest/download/OpenClaw%20Installer_0.1.0_universal.dmg
```

**更新 README.md：**

在本地修改 README.md 中的下载链接：

```markdown
[⬇️ 下载 OpenClaw-Installer.dmg](https://github.com/你的用户名/OpenClaw-installer/releases/latest/download/OpenClaw%20Installer_0.1.0_universal.dmg)
```

然后提交并推送：

```bash
git add README.md
git commit -m "Update download link"
git push
```
