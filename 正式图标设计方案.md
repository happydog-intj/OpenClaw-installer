## 📥 如何使用

### 方法1: 自动脚本（推荐）

```bash
cd ~/Documents/Github/OpenClaw-installer/src-tauri/icons
./选择图标.sh
```

按提示输入 1-4 选择方案，脚本会自动：
1. 复制选中的图标为 `icon.png`
2. 生成所有平台的图标文件
3. 提示下一步操作

### 方法2: 手动操作

```bash
cd ~/Documents/Github/OpenClaw-installer/src-tauri/icons

# 1. 预览所有方案
open icon-formal-v1.png
open icon-formal-v2.png
open icon-formal-v3.png
open icon-formal-v4.png

# 2. 选择一个（例如方案3）
cp icon-formal-v3.png icon.png

# 3. 生成所有尺寸
cd ../..
npx @tauri-apps/cli icon src-tauri/icons/icon.png

# 4. 重新构建
pnpm tauri build --target universal-apple-darwin
```
